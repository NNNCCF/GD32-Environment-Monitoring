# GD32 FreeRTOS 多传感器监控系统

## 项目简介

这是一个基于GD32F10x微控制器和FreeRTOS实时操作系统的多传感器监控项目。系统集成了温湿度传感器(DHT11)、气体传感器(MQ2)、OLED显示屏、按键和LED指示灯，通过UART串口进行数据输出和命令控制。

## 功能特性

- **多任务实时系统**: 基于FreeRTOS实现多任务并发执行
- **温湿度监测**: DHT11传感器实时采集环境温度和湿度数据
- **气体浓度检测**: MQ2传感器监测气体浓度并转换为百分比显示
- **数据显示**: 0.96寸OLED屏幕实时显示传感器数据
- **串口通信**: UART输出传感器数据，支持接收命令控制LED
- **按键控制**: 物理按键可以切换LED状态
- **LED指示**: LED灯具有自动闪烁和手动控制功能

## 硬件要求

- GD32F10x系列微控制器 (推荐GD32F103C8T6)
- DHT11温湿度传感器
- MQ2气体传感器模块
- 0.96寸OLED显示屏 (I2C接口)
- 按键开关
- LED指示灯
- USB转TTL串口模块 (用于调试和通信)

## 接线说明

### 传感器连接

| 模块 | 引脚 | GD32引脚 | 说明 |
|------|------|----------|------|
| DHT11 | DATA | PB8 | 温湿度传感器数据引脚 |
| MQ2 | AO | PA0 | 气体传感器模拟输出 (ADC输入) |
| MQ2 | VCC | 5V | 必须接5V电源 |
| MQ2 | GND | GND | 接地 |
| OLED | SCL | PB6 | I2C时钟线 |
| OLED | SDA | PB7 | I2C数据线 |
| OLED | VCC | 3.3V | OLED电源 |
| OLED | GND | GND | OLED接地 |

### 控制接口

| 模块 | 引脚 | GD32引脚 | 说明 |
|------|------|----------|------|
| LED1 | 阳极 | PC13 | LED指示灯 (低电平点亮) |
| KEY1 | 信号 | PE6 | 按键输入 (低电平有效) |

### 串口通信

| 功能 | GD32引脚 | 说明 |
|------|----------|------|
| UART_TX | PA9 | 串口发送 (连接USB转TTL的RX) |
| UART_RX | PA10 | 串口接收 (连接USB转TTL的TX) |

### 电源连接

- GD32F10x: VCC=3.3V, GND
- 所有传感器GND连接到同一GND
- MQ2传感器VCC单独接5V电源

## 软件环境

- **开发环境**: Keil MDK 或其他支持ARM Cortex-M3的IDE
- **操作系统**: FreeRTOS v10.x
- **编译器**: ARMCC 或 GCC
- **调试工具**: J-Link, ST-Link 或其他支持GD32的调试器

## 烧录步骤

1. **安装开发环境**
   - 下载并安装Keil MDK (版本5.0或以上)
   - 安装GD32F10x设备支持包

2. **打开项目**
   - 使用Keil打开 `test.uvprojx` 项目文件

3. **配置编译选项**
   - 选择合适的GD32F10x型号 (如GD32F103C8T6)
   - 配置调试器 (J-Link或ST-Link)

4. **编译项目**
   - 点击"Build"或按F7编译项目
   - 确保无编译错误

5. **烧录程序**
   - 连接调试器到GD32开发板
   - 点击"Download"或按F8烧录程序
   - 等待烧录完成

6. **断开调试器**
   - 烧录完成后断开调试器连接

## 使用方式

### 系统启动

1. 给GD32开发板上电
2. 系统自动初始化所有外设
3. OLED屏幕显示初始化信息
4. LED开始自动闪烁

### 数据查看

- **OLED显示**: 屏幕显示当前温度、湿度和气体浓度
- **串口输出**: 连接USB转TTL到电脑，使用串口助手查看数据
  - 波特率: 115200
  - 数据位: 8
  - 停止位: 1
  - 校验位: 无

### 串口命令控制

通过串口发送以下命令控制LED:

- `LED:ON` - 点亮LED
- `LED:OFF` - 熄灭LED
- `LED:TOGGLE` - 切换LED状态

### 按键操作

- 按下KEY1按键可以切换LED的亮灭状态

### 传感器数据

- DHT11: 每2秒采集一次温湿度数据
- MQ2: 每1秒采集一次气体浓度数据
- 数据通过UART实时输出，并在OLED上显示

## 故障排除

### 常见问题

1. **OLED不显示**
   - 检查I2C接线是否正确 (PB6=SCL, PB7=SDA)
   - 确认OLED模块供电正常 (3.3V)

2. **传感器无数据**
   - DHT11: 检查PB8接线，确认传感器供电
   - MQ2: 检查PA0接线，确认AO输出连接正确，VCC接5V

3. **串口无输出**
   - 检查PA9(TX)和PA10(RX)接线
   - 确认串口助手波特率设置为115200

4. **按键无响应**
   - 检查PE6接线，确认按键电路正常

5. **编译错误**
   - 确认所有头文件路径正确
   - 检查FreeRTOS配置是否匹配硬件

### 调试建议

- 使用串口助手监控系统运行状态
- 检查传感器供电电压是否正确
- 确认所有接线无短路或虚焊
